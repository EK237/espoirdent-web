Liaison Numérique Directe

Comment connecter votre scanner de cabinet à EspoirDent ?

Configurez vos exports d'empreintes optiques en moins de 2 minutes pour un transfert automatique sécurisé.

Medit Link

1. Ouvrez l'application Medit Link.

2. Accédez à la section "Partner Search".

3. Recherchez le laboratoire "EspoirDent".

4. Cliquez sur "Request Partnership". Vos scans nous parviendront directement.

3Shape Communicate

1. Connectez-vous sur votre portail 3Shape.

2. Allez dans l'onglet "Connections" puis "Laboratories".

3. Ajoutez notre email : mrk@dentova.xyz.

4. Validez. Nous apparaîtrons dans votre liste de laboratoires d'envoi.

Align/iTero

1. Contactez le support iTero pour lier votre scanner.

2. Transmettez notre code laboratoire officiel : ESPOIRDENT-YDE.

3. Vos empreintes iTero seront routées automatiquement vers nos concepteurs.
